# SAGE

Smart Assessment and Guidance Engine (SAGE).

## Project Structure

- `backend/`: FastAPI application with agents and LangGraph pipeline.
- `frontend/`: React application (to be initialized).

## Setup

1.  Navigate to `backend/` and install dependencies: `pip install -r requirements.txt`.
2.  Set up your environment variables in `.env` (copy from `.env.example`).
3.  Run the backend: `uvicorn main:app --reload`.
